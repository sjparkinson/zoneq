# zoneq

[![Build Status](https://travis-ci.org/sjparkinson/zoneq.svg?branch=master)](https://travis-ci.org/sjparkinson/zoneq)

Query zone files.

```
zoneq

Usage:
  zoneq <pattern> [<file>...]
  zoneq -h | --help
  zoneq --version

Example: zoneq *.example.com example.zone

Options:
  -h --help     Show this screen.
  --version     Show version.
```
